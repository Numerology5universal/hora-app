What is Hora?
• Hora = Hour (1 ghanta).
• “Hour” word actually Hora se hi derive hua hai.

Numerology vs Astrology Hora
Astrology (Jyotish)
• Har din ka har grah ka Hora hota hai .
• Duration change hota hai (1 hr, 1.5 hr, location dependent)

Numerology
• Hora fixed hota hai.
• ❌ No date, ❌ No month, ✅ Only time based system.

Time System
o 1 PM = 13
o 2 PM = 14 … up to 24 (Full day = 1 to 24 hours blocks)

Hora Table Structure
• Rows → Hours (1–24)
• Columns → Days (Sunday–Saturday)
• Har block mein ek number assigned hota hai

How to Use Hora
1. Day choose karo (Monday–Sunday).
2. Time choose karo (jaise 7–8 PM).
3. software find the intersection.
4. Jo number mile → wahi Hora number.

Important Rule
👉 Hora hour block basis par kaam karta hai
• 7:00 – 7:59 → same block (7 wala)
• 7:45 → still 7 block
• 8:00 → next block start
✔️ Ignore minutes
✔️ Only hour range dekho

Examples :
Eg : - 1 - Day: Saturday - Time: 5 PM → Table se number = 9
Eg : - 2 - Day: Tuesday - Time: 7 PM → Use 7–8 block
Eg : - 3 - Day: Monday - Time: 8:15 PM → Use 8–9 block (NOT 7)
